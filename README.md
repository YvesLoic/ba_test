# ba_test
 Projet de test pour Bridge Africa
# Procédure de test
 - installer les dépendances
~~~~
    composer install
~~~~
 - créer la bd mysql
~~~~
    php artisan create:db
~~~~
 - créer les tables en bd
~~~~
  php artisan migrate --seed
~~~~
 - démarrer le projet
~~~~
    php artisan serve
